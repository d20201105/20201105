# 위탁판매가 뭔가요?

# 쿠팡 파트너스

# 리다이렉트가 뭔가요?

한국 인터넷 사용자의 많은 수가 네이버 검색을 이용합니다.

쿠팡 파트너스와 같은 제휴 마케팅 활동을 하시는 분들은 검색 확률을 높이기 위해 네이버 블로그에 콘텐츠를 업로드 하는 것을 선호합니다.

네이버에서는 기계적으로 만들어지는 엄청난 분량의 제휴 마케팅 페이지를 달가워하지 않습니다.

몇몇의 제휴 마케팅 커뮤니티들에서 네이버 블로그를 빠져나가는 링크가 쿠팡 파트너스의 도메인인 경우 블로그가 저품질 처리될 수 있다고 알려져 있습니다.

이를 피하고자 다른 도메인의 페이지로 링크를 걸고 그 페이지에 사용자가 도달했을 때 쿠팡 페이지로 이동시키는 방식을 사용합니다.

이 때 다른 페이지로 이동시키는 방법의 이름이 리다이렉트입니다.

# 매출이 안 떠요

매출은 매출이 발생한 다음날 오후에 확인할 수 있습니다.

# 수익이 마이너스로 잡혀요

제휴 링크를 통해 물건을 주문한 쿠팡 사용자가 주문을 취소한 경우입니다. 환불이 되므로 그만큼 수익이 떨어집니다.

# 도메인은 어디서 사나요?

현재 커뮤니티에서 hosting.kr의 1년 9,800원 서비스를 추천하고 있습니다.
